# Global Layoffs Analysis (2020–2025)
Data Analytics project analyzing global layoffs trends using Python and Power BI

## 📊 Project Overview
This project analyzes global layoffs data to identify trends across industries, companies, and countries.

## 🎯 Objectives
- Identify industries with highest layoffs
- Analyze layoffs by country
- Study trends over time
- Understand relationship between funding and layoffs

## 🛠 Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Power BI

## 📁 Dataset
Dataset collected from Kaggle (Layoffs.fyi)
https://drive.google.com/file/d/1TI-YxbT6b7vaIwsaZVSRZb5Rd-EXe_JF/view?usp=sharing

## 📈 Key Insights
- Technology industry has highest layoffs
- Layoffs increased over time
- Few companies contribute to large layoffs
- Funding does not prevent layoffs
